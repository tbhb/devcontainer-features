# zsh-autosuggestions

Installs [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions), an autosuggestions plugin for zsh inspired by the Fish shell.

## Usage

```json
{
  "features": {
    "ghcr.io/tbhb/devcontainer-features/zsh-autosuggestions:1": {}
  }
}
```

## Configuration

After installation, the plugin is automatically sourced in your `.zshrc`. Suggestions appear as you type based on your command history.

Press the right arrow key or `End` to accept a suggestion.
